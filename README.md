# Demo app for state-generator

This app is a demo for:

https://www.npmjs.com/package/state-generator

## App

This was created with the following steps:

```
yarn create react-app demo-app-state-generator
```

```
yarn add \
    state-generator \
    redux \
    redux-saga \
    react-redux \
    axios@1.1.3 \
    reactstrap \
    bootstrap
```

## Run the application

```shell
yarn install
yarn start
```
